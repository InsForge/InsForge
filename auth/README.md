# Insforge Auth

Built-in authentication UI for Insforge users.

This is an independent mini-app that provides authentication pages for Insforge users including:
- Sign In
- Sign Up
- Email Verification
- Forgot Password
- Reset Password

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Email/Password authentication
- OAuth provider support (Google, GitHub, Discord, etc.)
- Email verification (code and link methods)
- Password reset flow
- Cross-tab communication for auth events
- Responsive design with dark mode support
