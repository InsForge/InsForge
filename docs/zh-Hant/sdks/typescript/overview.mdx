InsForge TypeScript SDK 為 JavaScript 和 TypeScript 應用程式提供了一個類型安全的用戶端，用於與 InsForge 服務互動。

## 快速開始

```typescript
import { createClient } from '@insforge/sdk';

const insforge = createClient({
  baseUrl: 'https://your-app.insforge.app',
  anonKey: 'your-anon-key'  // Optional: for public/unauthenticated requests
});
```

## 功能

- **Database** - 使用 PostgREST 進行類型安全的 CRUD 操作
- **Authentication** - 電子郵件/密碼和 OAuth 身分驗證
- **Storage** - 檔案上傳、下載和管理
- **Functions** - 叫用無伺服器邊緣函數
- **AI** - 聊天完成和圖像生成
- **Realtime** - 基於 WebSocket 的 pub/sub 訊息傳遞
- **Payments** - 提供商特定的 Stripe 和 Razorpay 支付助手
