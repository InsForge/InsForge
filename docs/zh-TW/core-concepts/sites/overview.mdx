使用 InsForge Sites 來部署屬於您的專案的面向瀏覽器的應用程式。InsForge CLI 透過 InsForge 上傳您的前端源，它建立一個 Vercel 生產部署。儀表板追蹤 URL、狀態、部署歷史、環境變數和域。

[Traditional Chinese Taiwan translation...]

## 下一步

- 設定 [CLI](/quickstart) 並連接您的專案。
- 從儀表板或使用 `npx @insforge/cli deployments env set` 新增瀏覽器安全環境變數。
- 執行 `npx @insforge/cli deployments deploy ./frontend`。
