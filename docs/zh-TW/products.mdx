InsForge 提供十個產品，每個產品都透過相同的 REST 表面、相同的 JWT 身份驗證、相同的 MCP 伺服器和相同的 SDK 公開。

  </Card>

  <Card title="身份驗證" icon="shield" href="/core-concepts/authentication/overview">

  </Card>

  <Card title="儲存" icon="folder" href="/core-concepts/storage/overview">

## 它們如何配合

每個產品共享相同的身份驗證身分識別、相同的列級安全原則和相同的 AI 代理 MCP 表面。透過**身份驗證**登入的使用者獲得 JWT，**資料庫**、**儲存**、**即時**和**邊緣函數**層都使用它來強制相同的存取規則。
