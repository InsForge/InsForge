The InsForge Python SDK provides a Python client for interacting with InsForge services from server-side scripts, APIs, data pipelines, and automation.

## Installation

```bash
pip install insforge
```

For Realtime support (WebSocket pub/sub):

```bash
pip install "insforge[realtime]"
```

## Quick Start

```python
from insforge import create_client

client = create_client(
    base_url="https://your-app.region.insforge.app",
    anon_key="your-anon-key"
)
```

## Features

- **Database** - CRUD operations with a fluent query builder (PostgREST-compatible)
- **Authentication** - Email/password, OAuth, profile management
- **Storage** - File upload, download, and management
- **Functions** - Invoke serverless edge functions
- **AI** - Chat completions (streaming + non-streaming) and image generation
- **Realtime** - WebSocket-based pub/sub messaging
