The InsForge Go SDK provides a Go client for interacting with InsForge services from server-side applications, APIs, CLIs, and microservices.

## Installation

```bash
go get github.com/InsForge/InsForge/sdks/go/insforge@latest
```

## Quick Start

```go
package main

import (
    "fmt"
    "github.com/InsForge/InsForge/sdks/go/insforge"
)

func main() {
    client := insforge.CreateClient(insforge.Config{
        BaseURL: "https://your-app.region.insforge.app",
        AnonKey: "your-anon-key",
    })

    session, err := client.Auth.SignInWithPassword("user@example.com", "password")
    if err != nil {
        panic(err)
    }
    fmt.Println("Logged in as:", session.User.Email)
}
```

## Features

- **Database** - Type-safe CRUD operations with a fluent query builder
- **Authentication** - Email/password and OAuth authentication
- **Storage** - File upload, download, and management
- **Functions** - Invoke serverless edge functions
- **AI** - Chat completions (streaming + non-streaming) and image generation
- **Realtime** - WebSocket-based pub/sub messaging
- **Thread-safe** - Concurrent token management with `sync.RWMutex`
