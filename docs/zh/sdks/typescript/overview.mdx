InsForge TypeScript SDK 为 JavaScript 和 TypeScript 应用程序提供了一个类型安全的客户端，用于与 InsForge 服务交互。

## 快速开始

```typescript
import { createClient } from '@insforge/sdk';

const insforge = createClient({
  baseUrl: 'https://your-app.insforge.app',
  anonKey: 'your-anon-key'  // Optional: for public/unauthenticated requests
});
```

## 功能

- **Database** - 使用 PostgREST 进行类型安全的 CRUD 操作
- **Authentication** - 电子邮件/密码和 OAuth 身份验证
- **Storage** - 文件上传、下载和管理
- **Functions** - 调用无服务器边缘函数
- **AI** - 聊天完成和图像生成
- **Realtime** - 基于 WebSocket 的 pub/sub 消息传递
- **Payments** - 提供商特定的 Stripe 和 Razorpay 支付助手
