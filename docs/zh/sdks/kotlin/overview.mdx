## 功能

- **数据库** - 使用 Kotlin 序列化实现类型安全的 CRUD 操作
- **身份验证** - 电子邮件/密码和 OAuth 身份验证
- **存储** - 文件上传、下载和管理
- **AI** - 聊天完成和图像生成
- **实时** - 基于 WebSocket 的发布/订阅消息传递
- **协程** - 完整的 Kotlin 协程支持
- **Jetpack Compose** - 可组合函数和状态管理

## 平台支持

| 平台 | 支持 |
|------|------|
| Android | 5.0+ (API 21+) |
| JVM | Java 11+ |
| Kotlin 多平台（即将推出） | iOS、macOS、JS（实验性） |
