## 概览

InsForge MCP 服务器为您的 AI 编码助手提供对您的后端的直接访问：数据库查询、模式管理、存储等。

按照下面针对您的 AI 客户端的说明进行操作。看不到您的工具？浏览[代理目录](https://insforge.dev/agents)以获取完整的经过测试的集成列表。

## 先决条件

- 一个**AI 编码助手**（Cursor、Claude Code、GitHub Copilot 等）
- 一个 **InsForge 项目** - 如果您还没有[创建一个](https://insforge.dev)

## 远程 MCP

开始使用的最快方式是远程 MCP。单个命令会尝试安装，并会尝试处理身份验证和项目绑定，但如果自动流程无法完成，可能仍会提示您进行手动身份验证或项目绑定步骤。

在您的终端中运行以下命令：

```bash
npx add-mcp https://mcp.insforge.dev/mcp
```

<Note>
  `add-mcp` 命令适用于大多数编码代理，但不是全部。目前它支持：
  Claude Code、Claude Desktop、Codex、Cursor、Gemini CLI、Goose、GitHub Copilot、OpenCode、VS Code、Zed。
</Note>

运行该命令后，系统将自动为您的编码代理生成配置文件。

### 连接和绑定项目

某些 MCP 客户端会在设置期间自动提示您登录，而其他客户端可能需要手动身份验证或项目绑定步骤。无论如何，身份验证都会打开一个浏览器窗口，您可以在其中登录到您的 InsForge 帐户并向 MCP 客户端授予组织/项目访问权限。

### 验证安装

要验证连接，在您的 AI 编码助手中启动新的聊天会话并发送此提示：

```
I'm using InsForge as my backend platform, call InsForge MCP's fetch-docs tool to
learn about InsForge instructions.
```

如果连接成功，您会在 InsForge 仪表板（右上角）看到 **MCP 已连接**状态指示器。

![MCP 已连接](/images/mcp-setup/mcp-connected.png)
