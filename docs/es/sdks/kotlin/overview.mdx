El SDK de Kotlin de InsForge proporciona un cliente Kotlin nativo para interactuar con los servicios de InsForge desde aplicaciones Android y proyectos Kotlin Multiplataforma.

<KotlinSdkInstallation />

## Características

- **Base de datos** - Operaciones CRUD seguras de tipos con serialización Kotlin
- **Autenticación** - Autenticación por correo electrónico/contraseña y OAuth
- **Almacenamiento** - Carga, descarga y gestión de archivos
- **IA** - Finalizaciones de chat y generación de imágenes
- **Tiempo real** - Mensajería pub/sub basada en WebSocket
- **Corrutinas** - Soporte completo para corrutinas de Kotlin
- **Jetpack Compose** - Funciones componibles y gestión de estado

## Compatibilidad de plataformas

| Plataforma | Compatibilidad |
|------------|----------------|
| Android | 5.0+ (API 21+) |
| JVM | Java 11+ |
| Kotlin Multiplataforma (próximamente) | iOS, macOS, JS (experimental) |
