Los Pagos de InsForge son específicos del proveedor. Stripe usa Checkout alojado y Portal de Facturación. Razorpay usa órdenes y suscripciones nativas del proveedor más Razorpay Checkout. El SDK de TypeScript expone módulos de proveedor separados para ambos.

## Configuración Compartida

```typescript
import { createClient } from '@insforge/sdk';

const insforge = createClient({
  baseUrl: 'https://your-project.insforge.app',
  anonKey: 'your-anon-key'
});
```

Las llamadas de tiempo de ejecución de pagos requieren un token de usuario de InsForge. El pago único anónimo puede usar un token anónimo de InsForge. Las claves secretas del proveedor deben mantenerse en el lado del servidor y configurarse a través de Panel -> Pagos -> Configuración, CLI o API de administración.

## Cumplimiento

Para ambos proveedores, crea tablas propiedad de la aplicación como `public.orders`, `public.credit_ledger` o `public.team_entitlements`, luego rellena desde eventos de webhook del proveedor verificados en `payments.webhook_events`.

`payments.transactions` es una proyección de panel/informes. No la uses como flujo de trabajo empresarial principal.
