El SDK de TypeScript para InsForge proporciona un cliente seguro en tipos para interactuar con los servicios de InsForge desde aplicaciones JavaScript y TypeScript.

## Inicio rápido

```typescript
import { createClient } from '@insforge/sdk';

const insforge = createClient({
  baseUrl: 'https://your-app.insforge.app',
  anonKey: 'your-anon-key'  // Optional: for public/unauthenticated requests
});
```

## Funcionalidades

- **Database** - Operaciones CRUD seguras en tipos con PostgREST
- **Authentication** - Autenticación por correo electrónico/contraseña y OAuth
- **Storage** - Carga, descarga y gestión de archivos
- **Functions** - Invoca funciones serverless de edge
- **AI** - Completamiento de chat y generación de imágenes
- **Realtime** - Mensajería pub/sub basada en WebSocket
- **Payments** - Asistentes de pagos Stripe y Razorpay específicos del proveedor
