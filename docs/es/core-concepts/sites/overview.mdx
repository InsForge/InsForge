Utilice InsForge Sites para enviar la aplicación orientada al navegador que pertenece a su proyecto. El InsForge CLI carga su fuente frontend a través de InsForge, que crea un implementación de producción de Vercel. El panel rastrea la URL, estado, historial de implementación, variables de entorno y dominios.

[Spanish translation...]

## Próximos pasos

- Configure el [CLI](/quickstart) y conecte su proyecto.
- Agregue variables de entorno seguras para el navegador desde el panel o con `npx @insforge/cli deployments env set`.
- Ejecute `npx @insforge/cli deployments deploy ./frontend`.
