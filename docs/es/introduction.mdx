InsForge proporciona a su agente de codificación Base de datos, Autenticación, Almacenamiento, Tiempo real, Funciones de borde, Sitios y una Puerta de enlace de modelo para completar aplicaciones de pila completa de extremo a extremo.

## Conectar primero

Configuración requerida. Elija una ruta para vincular su entorno a InsForge antes de comenzar a construir. Comience con la CLI para una configuración universal basada en terminales; agregue MCP si su editor lo admite (Claude Code, Cursor, Windsurf, Codex). Puede ejecutar ambos lado a lado.

<CardGroup cols={2}>
  <Card title="Conectar vía CLI" icon="terminal" href="/quickstart">
    `npx @insforge/cli login` y vincule un proyecto. Funciona en cualquier terminal, MCP no es necesario.
  </Card>
  <Card title="Conectar vía MCP" icon="plug" href="/mcp-setup">
    Conecte el cliente MCP de su editor para que el agente obtenga esquemas en vivo, registros y contexto del proyecto.
  </Card>
</CardGroup>

¿Buscando instrucciones para una herramienta específica? Explore el [directorio de agentes](https://insforge.dev/agents) para encontrar pasos de configuración probados para Claude Code, Cursor, Windsurf, Codex y más.
