Una colección de ejemplos y guías prácticas para construir con InsForge - la forma más rápida de construir aplicaciones de pila completa con PostgreSQL, autenticación, almacenamiento, IA y funciones sin servidor.

## Inicio rápido

Para comenzar con cualquier ejemplo en este libro de recetas:

1. **Explore ejemplos** - Encuentre el marco o caso de uso que se adapte a sus necesidades
2. **Siga la guía** - Cada ejemplo incluye indicaciones de IA e instrucciones de configuración
3. **Obtenga el código** - Vea lo que genera la IA para usted
4. **Construir y personalizar** - Utilice los ejemplos como puntos de partida para sus proyectos
